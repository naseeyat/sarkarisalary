üìä Selection Process

STAGE 1: Tier-1 Computer Based Test (CBT)

  • Objective type questions
  • Total Marks: 100
  • Duration: 90-120 minutes (varies by post)
  • Subjects: General Knowledge, General English, Reasoning, Subject Knowledge
  • Negative Marking: 0.25 marks for each wrong answer

STAGE 2: Tier-2 CBT

  • Objective/Descriptive type
  • Total Marks: 100
  • Duration: 120 minutes
  • Subject-specific questions
  • Only for qualified Tier-1 candidates

STAGE 3: Interview / Skill Test

  • Teaching Posts: Demo Class + Interview (20-50 marks)
  • Non-Teaching Posts: Skill Test + Interview (20-50 marks)
  • Only for shortlisted Tier-2 qualified candidates
Final Merit: Based on combined performance in Tier-2 + Interview/Skill Test
